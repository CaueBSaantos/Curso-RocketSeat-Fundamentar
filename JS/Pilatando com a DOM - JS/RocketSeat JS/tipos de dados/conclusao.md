# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data types
    * Primitive /  Primitive value 
    * Structural 
    * Structrual Primitive

## Primitivos 

* String
* Number
* Boolean 
* Undefined
* Symbol 
* BigInt

## Estruturas 

* Object 
    * Array 
    * Map
    * Set
    * Date
    *...
* Function 

## Primirivo Estrutural / Structural Root Primitive 

* null
